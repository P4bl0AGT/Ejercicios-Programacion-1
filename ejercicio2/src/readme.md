## Ejecución del ejercicio 2

Para resolver y ejecutar el problema correspondiente al **Ejercicio 2: Simulador de carrera en la Ruta 64**, se debe ejecutar el archivo:

`CarreraRuta64.java`

Este archivo contiene el método principal (`main`), que inicia el programa y crea los 3 hilos correspondientes a los vehículos de la carrera.  
El archivo `Vehiculo.java` actúa como clase de apoyo y es utilizado internamente por `CarreraRuta64.java`, por lo que no se ejecuta de manera independiente.
