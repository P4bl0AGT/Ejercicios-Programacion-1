## Ejecución del ejercicio 1

Para resolver y ejecutar el problema correspondiente al **Ejercicio 1: Monitor de procesos en planta industrial**, se debe ejecutar el archivo:

`IndustrialPlant.java`

Este archivo contiene el método principal (`main`), que inicia el programa y crea los 10 hilos correspondientes a los sensores de temperatura.  
El archivo `Sensor.java` actúa como clase de apoyo y es utilizado internamente por `IndustrialPlant.java`, por lo que no se ejecuta de manera independiente.
