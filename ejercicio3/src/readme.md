## Ejecución del ejercicio 3

Para resolver y ejecutar el problema correspondiente al **Ejercicio 3: Diferencia entre start() y run() en el servidor**, se debe ejecutar el archivo:

`Server.java`

Este archivo contiene el método principal (`main`), que inicia el programa y permite probar la diferencia entre llamar directamente al método `run()` y utilizar `start()` para crear hilos independientes.  
El archivo `TareaProcesamiento.java` actúa como clase de apoyo y es utilizado internamente por `Server.java`, por lo que no se ejecuta de manera independiente.
